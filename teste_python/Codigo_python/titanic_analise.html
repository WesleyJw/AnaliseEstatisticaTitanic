Valores faltantes¶

O tratamento de valores faltantes parte crucial para qualquer anlise, em geral exitem diversas formas de trat-los, essas, podem ser simples, como substituir NaN's pela mdia, mediana, moda ou simplesmente a excluso dessa instncia, ou pode-se optar por abordagens mais complexas, com uso de modelos de regresso os de machine learn simples ou multivariados para estimativa de valores faltantes. Antes de comear a preencher NaN's importante verificar se essa varivel importante para o estudo e quais outras variveis podem ser utilizadas para estimar um valor faltante. Existem diferentes formas de imputao de valores faltantes, algumas so:

Imputao simples

Existindo poucos valores NaN's na varivel estudada podemos substitu-lo utilizando alguma medida de tendncia central, uma forma mais simples e rpida.

Imputao simples com regresso

Neste mtodo so considerados os valores das demais caractersticas para estimar o valor faltante. Isto pode ser feito baseado em modelos de regresso logstico (binria), multinomial ou linear.

Imputao mltipla

A Imputao Mltipla uma tcnica para analisar bancos de dados onde algumas entradas so faltantes. A aplicao dessa tcnica requer trs passos: imputao, anlise e agrupamento.

O banco estudado possui valores faltantes nas variveis Age, Cabin e Embarked. Calculando o total de NaN podemos identificar a melhor forma de preencher esses valores. Assim, calculamos a soma de valores faltantes para cada varivel:

O nmero de mulheres que sobreviveram acentuadamente maior em relao aos homens. Podemos avaliar a associao existente entre variveis qualitativas realizando o teste $\chi^2$ "qui-quadrado". O princpio bsico deste teste comparar propores, ou seja, as possveis divergncias entre as frequncias observadas e esperadas para um evento. A hipteses que queremos testar so:

Hiptese nula: As frequncias observadas no so diferentes das frequncias esperadas. No existe diferena entre as frequncias (contagens) de sobreviventes por sexo. Assim, no existe associao entre os grupos, sobreviventes por sexo.

Hiptese alternativa: As frequncias observadas so diferentes das frequncias esperadas. Portanto existe diferena entre as frequncias. Assim, existe associao entre os grupos sobreviventes por sexo.

Para calcularmos o teste $\chi^2$ criamos uma tabela com as frequncias de sobreviventes por sexo:

Segundo os estatsticos Moretin e Bussab o mnimo de 5 e o mximo 15 classes o mais indicado para o resumo de qualquer varivel. Dessa forma, utilizaremos 11 classes para resumir a faixa etria de passageiros. Posteriomente, criamos a sequncia com os intervalos das classes e em seguida vamos criar uma nova coluna na nossa tabela com a indicao da faxa etria de cada passageiro. As classes representam os intervalos numricos em que a varivel quantitativa foi classificada. A amplitude da classe determinada por $\frac{max(Idade) - min(Idade)}{N. classes}$.

Resposta:¶

Em um primeiro instante, pode-se pensar que quanto maior o banco de dados, representados por um volume elevado de variveis descritivas de observaes, seja prefervel para a explicao de um fenmeno. Contudo, algumas variveis no acrescentam nehuma informao adicional para explicao do fenmeno, alm de um grande nmero de variveis aumentar a dimensionalidade dos dados gerando alto custo computacional e as vezes inserindo um vis ao modelo. Diante disso, importante verificar quais as variveis que mais contribuem para o estudo. A correlao entre as variveis dependentes pode trazer essa informao, lembrando que, variveis que apresentarem baixa correlao podem ser excludas do modelo e variveis com alta correlao entre si podem ser substituidas uma pela outra. Cabe ressaltar que quase todos os mtodos da estatstica classe para inferir contribuio de variveis so lineares, logo ao se trabalhar com problemas no lineares esses mtodos apresentam baixa eficincia. Uma alternativa fazer uso de modelos de machine learn como, rvores de decises, florestas aletorias e redes neurais.
